## BEACON Package

本仓库包含 DX-SMART 蓝牙 BEACON 模块的完整资料包，包含开发固件、用户手册、参数配置等资源。

---

## 📦 包内文件说明

### 1. CP28&CP33.rar
**包含 CP28 / CP33 平台蓝牙 BEACON 相关开发与用户资料**
- 开发固件（BEACON 功能实现）
- 用户使用指南
- 关键参数配置表
- 模块硬件接口定义
- 测试命令与调试说明

### 2. CP29.rar
**包含 CP29 平台蓝牙 BEACON 开发包**
- 对应平台的 BEACON 工程源码
- 编译与烧录步骤
- 官方应用层协议
- 客户化开发示例

### 3. DX-CP27&CP27-NFC.rar
**包含 CP27 及 CP27-NFC 版本蓝牙 BEACON 资料**
- 标准 BEACON 协议包
- NFC 配对与配置文件
- 多场景广播参数模板
- 低功耗优化方案
- 兼容不同芯片版本的固件

### 4. DX-CP32.rar
**包含 CP32 平台蓝牙 BEACON 完整包**
- 稳定版主固件
- 增强型功能固件（可选项）
- 信息包解析文档
- 模块通信协议与API说明
- 现场部署配置工具

---

## 🛠 使用说明
1. 下载对应平台的压缩包。
2. 解压后查看内部 PDF、文档、固件与工程文件。
3. 按开发指南进行烧录、配置与测试。
4. 如有问题，可查阅文档或联系邮箱获取支持。
